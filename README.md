# Minesweeper Helper (Chrome Extension)

Расширение для помощи в игре на https://minesweeper.online/.

## Возможности

- **Alt + Shift + M** — подсветить мины в закрытых клетках.
- **Alt + Shift + P** — подсчитать и подсветить вероятность мины в закрытых клетках.
- **Alt + Shift + S** — подсветить безопасные клетки.

Подсветка отображается поверх поля с градиентом: зелёный означает безопасно, красный — опасно.

## Режим «Эффективность»

Включается слайдером в настройках расширения. В этом режиме вероятности отображаются только
для клеток с высоким или низким риском, чтобы быстрее выбирать следующий ход.

## Установка

1. Откройте Chrome → `chrome://extensions`.
2. Включите «Режим разработчика».
3. Нажмите «Загрузить распакованное расширение» и выберите папку `extension`.

Работает только на `minesweeper.online`.
