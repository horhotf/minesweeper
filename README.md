# Minesweeper Helper (Chrome Extension)

Расширение для помощи в игре на https://minesweeper.online/.

## Возможности

- **Alt + Shift + 1** — подсветить мины в закрытых клетках.
- **Alt + Shift + 2** — подсчитать и подсветить вероятность мины в закрытых клетках.
- **Alt + Shift + 3** — подсветить безопасные клетки.
- **Alt + Shift + 4** — подсветить следующий эффективный ход.
- **Escape** — скрыть подсветку расширения.

Подсветка отображается поверх поля с градиентом: зелёный означает безопасно, красный — опасно.

## Режим «Эффективность»

Включается слайдером в настройках расширения. В этом режиме подсвечивается следующий
эффективный ход (самая безопасная клетка или гарантированно безопасная зона).

## Установка

1. Откройте Chrome → `chrome://extensions`.
2. Включите «Режим разработчика».
3. Нажмите «Загрузить распакованное расширение» и выберите папку `extension`.

Работает только на `minesweeper.online`.
